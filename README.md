
# Sistema de Inscripciones

Nuestro proyecto se basa en un sistema de inscipciones para talleres o cursos, que permite además de saber cuales son los cursos más demandados, cual es el nivel de conocimiento de los estudiantes. Puede también realizarse la eliminación de algún alumno inscripto en el sistema y maneja una lista de espera para los cursos en caso de encontrarse llenos.



## Instrucciones de uso

Para usarlo debemos inicializar el programa. Luego nos aparecerá un menú con las opciones de inscripciones a cursos y talleres en las primeras opciones y después otras revisiones como ver la lista de espera o eliminar algún alumno.
